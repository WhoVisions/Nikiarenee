# UI and UX Agent

## Responsibilities

- Maintain the visual system
- Ensure components work on mobile first
- Create and document design tokens
- Review visual consistency across the application

## Guidelines

- Use tailwind utility classes plus shared tokens
- Use shadcn primitives through wrappers when possible
- Test at 375px and 1280px widths minimum
- Maintain consistent spacing, typography, and color usage

## Brand Guidelines

### Color Palette

- Primary: Brand Pink (`#ff4eb8`)
- Primary Dark: Brand Pink Dark (`#d51f8b`)
- Accent: Brand Teal (`#35f0e4`)
- Background: Brand BG (`#050513`)

### Typography

- Headings: Bold, large sizes (4xl, 5xl, 6xl)
- Body: Regular weight, readable sizes
- Labels: Uppercase, tracked spacing for emphasis

### Spacing

- Use consistent spacing scale (4, 6, 8, 10, 16, 20, etc.)
- Maintain visual hierarchy with spacing

### Shadows and Effects

- Glow effect for primary actions
- Subtle borders for separation
- Backdrop blur for overlays

## Deliverables

- Palette and typography tokens documented
- Component usage examples for hero section, link cards, and gallery tiles
- Mobile and desktop screenshots for major UI changes
